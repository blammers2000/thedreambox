Batch Enrich JSON Files (add titles)

Select multiple JSON files. Each file can be:

  • An array of video ID strings: ["id1","id2"]
  • An array of objects with id (and optional title): [{"id":"id1","title":"..."}]
  • An object with numeric keys (like {"0":{"id":"id1"},"1":{...}}) – will be auto‑converted.

The tool will fill missing/unknown titles using the YouTube API and save enriched files.
